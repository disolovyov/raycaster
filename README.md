# raycaster

This is a software raycaster based on [Lode's raycasting tutorials](https://lodev.org/cgtutor/index.html).

## Running

To build and run:

```
cargo run --release
```

To build and run in the browser (unless I accidentally broke it):

```
cargo web start --release
```
